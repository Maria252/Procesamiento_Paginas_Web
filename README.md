# Workshop IA para Empresas
Taller práctico: Aplicación de Inteligencia Artificial en la Ruta Exportadora y la Segmentación de Mercados

Este repositorio contiene los recursos, notebooks, presentaciones y documentación asociados al taller.

## Ruta Exportadora y Uso de IA para la Segmentación de Mercados
Ruta Exportadora: Cómo identificar oportunidades de exportación, evaluar mercados y adaptar productos para una entrada exitosa.

Uso de IA para la Segmentación de Mercados: Aplicación de herramientas de inteligencia artificial para facilitar la toma de decisiones en el proceso exportador.

## Estructura de Casos y Subtemas

| Nº | Tema Principal | Responsable | Duración | Subtemas / Actividades |
|:--:|:----------------|:------------|:--------:|:----------------------|
| 1 | Entendimiento de los Prompts | Enrique | 1.5 horas | Introducción al diseño y uso de prompts efectivos en IA |
| 2 | Exploración de oportunidades en múltiples mercados / Uso de IA para revisar oportunidades | Enrique | 3 horas | Evaluación de demanda internacional, análisis de competencia, análisis de datos de TradeMap, evaluación automatizada del potencial de exportación |
| 3 | Identificación de oportunidades y debilidades del mercado / Uso de IA para el análisis DOFA | Nico | 3 horas | Análisis DOFA, uso de DeepResearch y Copilot, comparación de herramientas para análisis estratégico |
| 4 | Predicción del tamaño del mercado y oportunidades de crecimiento con IA | Nico | 3 horas | Modelos predictivos para demanda futura, análisis de series temporales, identificación de patrones de crecimiento |
| 5 | Acceso a un mercado específico / Automatización del análisis de normativas y tendencias | María Paula | 3 horas | Evaluación de requisitos arancelarios y no arancelarios, uso de GPT para interpretar normativas, automatización de extracción de documentos regulatorios |
| 6 | Adecuación del producto y tendencias del mercado / Scraping y automatización de datos | María Paula | 2 horas | Identificación de tendencias de consumo (Mintel, DeepSearch, Appi), uso de Insta Data Scraper, integración con herramientas como Zapier |


## Contenido del repositorio
/notebooks/: Jupyter Notebooks con los ejercicios y análisis realizados.

/src/: Scripts de apoyo, funciones personalizadas y automatizaciones.

/data/: Datos utilizados en los casos prácticos (cuando aplicable).

/docs/: Documentación técnica, presentaciones y recursos adicionales.

/presentations/: Archivos PPT de las exposiciones de cada caso.

## Notas adicionales
* Todos los documentos (PPTs, notebooks, scripts) serán centralizados en este repositorio para su fácil acceso.
* Se sugiere clonar el repositorio y seguir la estructura de carpetas para mantener el orden de trabajo.
